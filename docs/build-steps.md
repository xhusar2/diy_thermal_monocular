# Build Steps

## Prerequisites

- All parts from the [BOM](BOM.md)
- 3D printed body parts
- Soldering iron with heated insert tip
- Soldering iron with fine tip + solder
- Hex/Allen key set
- Tweezers
- Hot glue gun (optional, for securing wires)

---

## Step 1 — Assemble the Body

- 1.1 Install Heated Inserts into the holes in the main body part 8xM2
- 1.2 Install Heated inserts for picatinny mount 1xM5
- 1.3 Install Heated inserts for J arm mount 1xM6
